# Paper Review

> 컴퓨터 세상에 족적을 남긴 논문을 정리합니다.

### 설명

* 1주 1 논문: 일주일에 각자 1 논문을 정리합니다.
* 상호 보완: PR에 올라온 정리를 다른 사람은 보충해줍니다.

----

### PR
* 하나의 PR은 하나의 문서입니다.
* 하나의 PR은 여러 개의 Commits로 이루어집니다.

  1️⃣ [FEAT] 해당 알고리즘에 대한 설명

  2️⃣ [EDIT] 보충 설명, 피드백
----

### 일정

|  요일  |      Task      |
|:----:|:--------------:| 
| 월요일  |     논문 선정      |
| ~토요일 |   정리 PR 남기기    |
| 일요일  | 피드백 반영 후 Merge |
